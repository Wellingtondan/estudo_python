**1. Soma**
total_empregados = q_seguranca + q_docente + q_diretoria

**2. Subtração**

diferenca_salario

**3. media ponderada**
media = (q_seguranca*s_seguranca + q_docente*s_docente + q_diretoria*s_diretoria) / (total_empregados)
